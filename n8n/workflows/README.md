# n8n/workflows/

Workflows n8n exportados como JSON y versionados aquí.

## Convención de nombres
`[modulo]-[descripcion].json`

Ejemplos:
- `cobros-alerta-vencimiento.json`
- `ventas-confirmacion-pedido.json`
- `sync-error-notification.json`

## Cómo importar
1. Abrir n8n: http://localhost:5678
2. Workflows → Import from file
3. Seleccionar el .json de esta carpeta
